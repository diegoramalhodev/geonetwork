Deployment
##########

On this first step, we will learn how to make GeoNetwork run on your machine. The first step is optional and can be skip if you already have a geonetwork.war file. You can download a war file on `Sourceforge <https://sourceforge.net/projects/geonetwork/>`_. There is also a `dockerized version of GeoNetwork <https://hub.docker.com/_/geonetwork/>`_ available.
